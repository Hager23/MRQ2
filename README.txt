My Reno Quotes mobile gate app (Expo + expo-router).

Before building:
- Replace API_BASE_URL in lib/api.ts with your real backend URL.
- Add real PNG images in assets/ with these names:
  - icon.png
  - splash.png
  - adaptive-icon.png
  - favicon.png

Then:
- Run `npm install`
- Run `npx expo start` for local dev
- Use EAS Build for store builds.
